## Scene Layouts - Prototype
This folder contains the **non-interactive** prototype for scene layouts for our Plague Village game. 
The layout contained here specifically focuses on our first scene, **Intro/Arrival**, when the Plague Doctor first arrives at the village...

## Folder Structure
### Code
- `index.html` - HTML code for the scene.
- `script.js` - JavaScript code for the scene.
- `styles.css` - CSS Styles for the scene.
### Assets & Music
- `infectedVillage.mp3` - Contains the music for our scene (Music can be left off or turned on, depending on preferences).
- `introVillager.png` - NPC Character Asset for this scene.
- `plagueDoctor.png` - Plague Doctor (player) Asset.
- `villageBackground.png` - Village Background Asset.
### Markdown Files as Documentation
- `instructions.md` - Contains instructions for viewing the prototype in browser.
- `intro-arrival-story-script.md` - Contains the Intro/Arrival story for the scene.
- `sprint1-deliverables-tasks` - Contains **important tasks for the frontend team** to complete the frontend **Sprint 1 Deliverables**.
